Our project summary:

Group 6: Erik Dubec, Lee Taylor, Raphael J Vasquez, Yang Yang


Objectives:

  • Display complex economic data in an easy to understand and compare format!
  • Allow for end users to investigate how different economic data points relate to each other on the state level.

Site Key:

  • Map - This shows how state level changes in economic data compare to the national average. On this page, the more green something is, the better off a state is, the more red it is, the worse off the state is.
  • Plotly- This page shows how economic data relates to each other, and updated multiple charts at once.

Technical Requirements

  • Database: We used SQLite
  • Dashboard Page: The Plotly Page
  • Combination of: Leaflet and Plotly
  • Flask Server: Peforms mulitiple manipulations of BEA data to make data useable
  • Library we did not Cover: Popper.js (Toggle State and Regional Data)
  • Dataset Size: Well over 1,000 records. 10 times the requirement.
  • User Driven Interation: Drop down events and menus
  • Final Visualization: We had far more than three views!
